## 🧠 Overview

LoanXAI simulates how modern lenders evaluate loan applications using:

- Machine Learning-based risk prediction  
- Feature-level explainability (SHAP)  
- Hybrid rule-based safeguards  
- Real-world credit indicators (income, CIBIL, employment)

---

## ✨ Key Features

- 🔍 **Explainable AI** — Understand *why* a loan is approved/rejected  
- ⚖️ **Hybrid Decision System** — ML + rule-based logic  
- 📊 **Risk Score & Repayment Probability**  
- 💡 **Actionable Suggestions** for applicants  
- 🇮🇳 **India-focused logic** (CIBIL, NBFC-style evaluation)

---

## 🛠 Tech Stack

- **Backend:** Python (Flask)  
- **ML Model:** XGBoost  
- **Explainability:** SHAP  
- **Frontend:** HTML, CSS, JavaScript
